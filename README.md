# Welcome
I'm a technology professional who builds systems and solutions, and a gamer who builds worlds and experiences. This space is where I try to bring both to life. 

## About Me
Professionally, I am equal parts system engineer, analyst, administrator, and support staff. In recent years, I've undertaken more and more technology projects that are bigger, more development-focused work. 

In my personal life, I'm a tech enthusiast and avid gamer, and I administrate a full stack of Linux-based open source services across multiple platforms.

I have a mix of professional and personal projects I host here as I tinker, learn, and grow my skillset. 

### My Professional Projects
*Small but impactful IT tools for use in professional production environments*

- **Service Delivery Toolkit** - An extendable GUI application allowing IT teams to use scripts from a remote repository in real time
- **UpSync** - A GUI tool that allows IT teams to automatically update remote files based on their local versions

### My Personal Projects
*Tech projects for personal or public use*

- **Squared Circle Universe** - A wrestling simulation app which brings the concept of WWE games' "Universe Mode" to a fully customizable PC-based experience
- **Strategium** - An application for Warhammer 40,000 tabletop game which allows the creation and live updating of campaign battle maps
- **Brink SRD** - System resource document for BRINK, my original near-future tabletop role-playing game

### What I'm Exploring
- Flutter for cross-platform UI
- FastAPI for lightweight backends
- Privacy-respecting hosting and self-hosting models
